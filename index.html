Resolution method solver

Algorithm that proves unsatisfiability of set of formulas in predicate logic.

How to use:

Input the language symbols. Input formulas that you would like to prove are unsatisfiable. Click make transformations to transform the formulas to disjunct sets. Then click apply resolution.

Note that the method is semi-computable - if your formulas are satisfiable then the algorithm will never end.
