Resolution method solver

Algorithm that proves unsatisfiability of set of formulas in predicate logic.

How to use:

Input the language symbols. Input formulas that you would like to prove are unsatisfiable. Click make transformations to transform the formulas to disjunct sets. Then click apply resolution.The algorithm will try to find a step by step proof with step count less than the maximum.

Note that the method is semi-computable - if the provided formulas are satisfiable then the algorithm will fail to come up with a solution no matter the size of max steps.
